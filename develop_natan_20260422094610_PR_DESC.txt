# 🚀 Sugestão de Pull Request

**Commit Message Recomendada:**
```text
feat: adicionar ícone de executável no build e aviso de diff vazio na CLI
```

---

### Resumo
Esta PR traz melhorias para a experiência do usuário (UX) e para o empacotamento da aplicação. Agora a ferramenta fornece um feedback claro no terminal quando tenta-se gerar uma PR sem alterações no código, e o binário gerado passa a ter um ícone personalizado.

### Mudanças Técnicas
- **`README.md`**: Adicionada a flag `--icon=icon.ico` na instrução de build do `pyinstaller` para embutir o ícone no executável gerado.
- **`src/main.py`**: Implementada uma mensagem de aviso em amarelo utilizando `click.secho` que é disparada caso o retorno do `get_git_diff()` seja vazio ou nulo.

### Impacto
- **Feedback de CLI**: Os usuários agora entenderão imediatamente por que a geração do PR foi interrompida caso não tenham feito commit ou stage das mudanças.
- **Visual do Binário**: O executável standalone será gerado com uma identidade visual própria (`icon.ico`), garantindo um aspecto mais polido e profissional à ferramenta.
