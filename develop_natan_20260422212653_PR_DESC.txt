# 🚀 Sugestão de Pull Request

**Commit Message Recomendada:**
```text
feat: adiciona atalho -h para ajuda e aprimora documentação de uso da CLI
```

---

### Resumo
Este PR melhora a usabilidade e a experiência do desenvolvedor (DX) ao utilizar o GitPR CLI. Ele introduz o atalho `-h` para exibir o menu de ajuda e expande significativamente a documentação do projeto, esclarecendo o comportamento padrão da ferramenta e suas funcionalidades avançadas, tanto no código-fonte quanto no arquivo README.

### Mudanças Técnicas
* Inclusão do decorador nativo `@click.help_option('-h', '--help')` no arquivo `src/main.py` para permitir o uso de `-h`.
* Atualização dos textos de ajuda das flags `--review` e `--fullreview` na CLI para explicitar as bases de comparação (`git diff` e `origin/main`).
* Adição de explicação sobre o 'Comportamento Padrão' (sem passagem de flags) na docstring da função principal `cli`.
* Reestruturação da seção de uso no `README.md`, adicionando exemplos práticos para o 'Comportamento Padrão' e detalhando as 'Opções e Comandos Avançados' (`--commit`, `--review`, `--fullreview`, `--skill`, `-h`).

### Impacto
Os usuários terão muito mais facilidade para entender e operar a ferramenta diretamente no terminal ou lendo o repositório, garantindo um melhor proveito dos recursos de IA do GitPR para geração de Pull Requests, Code Review e mensagens de commit.
