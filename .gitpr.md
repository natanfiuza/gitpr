# Contexto do Projeto (GitPR Skill)

## 🎯 Sobre o Projeto
Este é um sistema GitPR. Ele é focado em gera decrição de pull request, faz review de código e gera frases de commit.

## 🏗️ Arquitetura e Tecnologias
- Linguagem principal: Python 3.13


## 📏 Regras de Negócio e Clean Code
1. **Nomenclatura**: Variáveis e métodos em `snake_case`, classes em `PascalCase`.
2. **Idioma**: Código em inglês ou português, mensagens em português.
3. **--commit**: A frase deve ser em português e refletir claramente a essência da mudança feita no código.
